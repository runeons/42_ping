# 42 ft_ping
    
#### Overview
This is a 42 project about recoding the ping command.

#### Options
- -?, --help
- -v, --verbose
- -c, --count
- -i, --interval
- -w, --timeout
- -t, --ttl
- -p, --pattern

#### Information
- requires superuser privileges
- ping version (GNU inetutils) 2.4 taken as a reference
- functional on Debian GNU/Linux 12

#### Usage
    make && ./ft_ping -?
